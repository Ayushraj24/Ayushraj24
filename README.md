## 📌 Featured Projects

### 🍕 Pizzeria Analytics Warehouse

Created a local analytics warehouse for pizzeria operations using order, customer, menu, inventory, vendor purchase, staff shift, and profitability data.

**Tech Used:** Python, SQL, DuckDB, Metabase, Docker  
**Repository:** [Pizzeria Analytics Warehouse](https://github.com/Ayushraj24/pizzeria-duckdb-metabase)

---

### 💳 PayFlow Fraud Analytics Lakehouse

Worked with 6.36M PaySim mobile-money transactions and organized the project into raw, bronze, silver, gold, quarantine, quality, metadata, and dashboard layers.

**Tech Used:** Python, DuckDB, Parquet, Streamlit, Altair, pytest  
**Repository:** [PayFlow Fraud Analytics Lakehouse](https://github.com/Ayushraj24/payflow-fraud-lakehouse)

---

### 🎵 Event-Driven Data Pipeline

Designed an event-driven ETL pipeline for music retail data using Snowflake and Apache Airflow with a Bronze-Silver-Gold structure.

**Tech Used:** Python, SQL, Snowflake, Apache Airflow  
**Repository:** [Event-Driven Data Pipeline](https://github.com/Ayushraj24/Event-Driven-Data-Pipeline-Music-retail-)

---

### 🏦 Credit Risk & Fraud Analysis

Performed exploratory data analysis on credit and fraud indicators and created Power BI dashboards to understand risk patterns, customer segments, and repayment behavior.

**Tech Used:** Python, Pandas, Power BI  
**Repository:** [Credit Risk & Fraud Analysis](https://github.com/Ayushraj24/Credit-Risk-Fraud-Analysis-EDA-BI-)
